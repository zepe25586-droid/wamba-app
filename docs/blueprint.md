# **App Name**: TontineManager

## Core Features:

- Member Management: Add, view, and manage members of the Tontine. The administrator (momojulinio1@gmail.com) has full access, while other members have read-only access.
- Archive Management: Create, update, and archive contributions. The administrator has sole modification rights.
- Contribution Tracking: Track each member's contribution history and current balance.
- Admin Authentication: Recognize the administrator (momojulinio1@gmail.com) upon login and grant elevated privileges.
- Reporting: Generate reports on Tontine activities, such as total contributions, balances, and historical data.
- Role-based Access Control: Verify if a user trying to access is momojulinio1@gmail.com or not to determine their priviledge on each function

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and stability.
- Background color: Light gray (#F5F5F5) for a clean, readable interface.
- Accent color: Soft green (#4CAF50) to highlight important actions and success.
- Body and headline font: 'PT Sans' for a modern, approachable, readable style.
- Use simple, consistent icons to represent actions and data categories.
- Use a clear, tabular layout for displaying member and contribution data.
- Subtle transitions and feedback animations to improve user experience.
